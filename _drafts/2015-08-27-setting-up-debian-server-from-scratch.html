1. Install Nginx server

$ apt-get install nginx

2. Check server status

$ service nginx status

If it is not running run it by

$ service nginx start

to check where the configuration files are located and if they are fine,

$ sudo /usr/sbin/nginx -t

if the nginx binary location is unknown, then try $ which nginx

3. Install PostgreSQL

$ apt-get install libpq-dev python-dev

$ apt-get install postgresql postgresql-contrib

to check status and start, stop and restart the server use service utility

To create a database with role along it to use in Django project,

$ su - postgres

the above command will log us into postgres user account,

First, create a database for testing,

$ createdb database_name

then create a role to associate with that database we've created.

$ createuser role_name -P

finally, grant the privileges for the DB to the role we've created,

$ psql

postgres=# GRANT ALL PRIVILEGES ON DATABASE database_name TO role_name;

Once the database and the respective role has been created, then we are ready to move on to Django.

###### END OF SERVER CONFIGURATION ######
